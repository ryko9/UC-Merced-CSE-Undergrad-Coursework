    • What is a Firewall?

    A firewall is a network security device which monitors incoming and outgoing network traffic and permits or blocks data packets which are based on a set of security rules. The purpose is to create a barrier between someones internal network and incoming from traffic from external sources which are ordered to block malicious traffice like virsuse and hackers. Firewall help keep viruses or malware away from the devices which are being used on the wifi and help stop the viruses. Firewall has been improved throughout the years and have created new ways on how the firewall can benefit the users.

    Different types of Friewalls.

    There are different types of firewalls that are used depending on the type of business or use of firewalls in someones home and business. The four different type of firrewalls are next- generation firewalls, Proxy firewalls, Network address translation(NAT) firewalls, and Stateful multilayer inspection (SMLI) firewalls.

  • Next-Generation firewalls- These combine a traditional firewall technology with additional functionality. This would be encrypted traffic inspection, intruion prevention systems, anit-virus, and many more. This would also include deep packet inspection. Another category which would be a distrubuted firewall as well. A distrubited firewall are host-resident security which help protect enterprise network's servers and end-user machines against unwanted intrusion.
  • Proxy firewalls- A filter network traffic at the application level. Unlkie basic firewalls, the proxy acts as an intermediary between two end systems.
  • Network address translation- The NAT allows multiple devices with independent netwok addresses to connect to the internet using a single IP address keeping individual IP addresses hidden.
  • Stateful multilayer inspection firewalls- This is a filter packet at the network, transport, and application layers, comparing them against known tursted packets. Like NGFW firewalls, SMLI also examine the entire packet and only allow them to pass if they pass each layer individually.
